🔥 Firebase Config คืออะไร?

Firebase คือบริการฐานข้อมูลของ Google ใช้เก็บข้อมูลรายจ่ายของคุณบน cloud ซิงค์ข้ามอุปกรณ์ได้ทันที และฟรีสำหรับการใช้งานส่วนตัว

1
สร้าง Firebase Project
ไปที่ console.firebase.google.com → คลิก "Add project" → ตั้งชื่อ → สร้าง
2
เปิด Firestore Database
เมนูซ้าย → Build → Firestore Database → Create database → เลือก Production mode → region: asia-southeast1
3
ตั้ง Rules (เปิดให้อ่านเขียนได้)
Firestore → แท็บ Rules → วางโค้ดนี้แล้วกด Publish:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
4
หา Config ที่ไหน?
⚙️ Project Settings (ซ้ายบน) → เลื่อนลงหา "Your apps" → คลิก </> Web → ตั้งชื่อ App → Register

จะได้ค่า 3 ตัวนี้:
apiKey"AIzaSy..."
projectId"my-project-12345"
appId"1:123456:web:abc..."
💡 ใช้ฟรีตลอด? ใช่ — Firestore มี free tier 1 GB storage + 50,000 reads/วัน เกินพอสำหรับใช้ส่วนตัว
